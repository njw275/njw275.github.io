Tuned is a web-based application created to recommend music to groups of users; the application runs on Node.js with the Express web application framework, but also includes a CouchDB server to store previously analyzed music and makes use of Socket.io for communication across client facing web pages. Tuned uses content-based filtering to choose songs aimed towards the favor of the entire group. Using the Spotify API, Tuned collects users' songs along with inputs from the users specifying their desired danceability, popularity, instrumentalness, and mood of songs. Once users are ready, Tuned gets the audio analysis from a Spotify API call for each song, and runs a k-means clustering algorithm on the array of timbre segments the API call returns. From this song analysis, Tuned finds a song from the collection of users' songs to use as the center point for its music recommendation. Along with the input preferences, Tuned uses the song chosen and creates a playlist tailored to the group. On the web-based application, Tuned shows the playlist along with previews of the songs; the application also creates the playlist in its entirety on all users' personal Spotify accounts.
